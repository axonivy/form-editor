# Form Editor (Prototype)

This repo contains the prototype of the web-based next gen form editor.

## Client

This prototype is built with React.

### Available Scripts

`pnpm run dev`: Start the dev server

### VsCode dev environment

#### Debug

Simply start the `Start Standalone` launch config to get debug and breakpoint support.

#### Run tests

To run tests you can either start a script above or start Playwright or Vitest with the recommended workspace extensions.
